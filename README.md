# Abu Al-Ghuwar: The Intelligent Interactive AI Bot 🤖

**Abu Al-Ghuwar** (أبو الغوار للرد المحتار) is a smart automation system built using **n8n** and **Google Gemini AI**. It's designed to handle customer interactions on Telegram with a touch of humor and high efficiency.

## 🚀 Features
- **Smart Intent Classification**: Uses Gemini AI to understand if the user is complaining, asking about prices, or just chatting.
- **Dynamic Responses**: Automatically sends different types of media (Text, GIFs, or Data updates) based on user mood.
- **Business Integration**: Automatically logs complaints or leads into **Google Sheets** for follow-up.
- **Humorous UX**: Breaking the ice with customers using funny animations (GIFs) to handle tough objections like "High Prices".

## 🛠️ Tech Stack
- **n8n**: The backbone of the automation workflow.
- **Google Gemini**: The "Brain" for NLP and sentiment analysis.
- **Telegram Bot API**: The primary interface for users.
- **Google Sheets**: For real-time data logging.

## 📦 Installation
1. Clone this repository.
2. Import the `workflow.json` file into your **n8n** instance.
3. Set up your credentials for Telegram, Google Sheets, and Gemini AI.
4. Hit **Execute** and let Abu Al-Ghuwar take over!
